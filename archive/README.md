# Archive

## Files

- **Features-old.tsx** - Original Features component (replaced by Features.tsx)
- **UseCases-old.tsx** - Original UseCases component (replaced by UseCases.tsx)

## Why Keep These?

These files are archived (not deleted) for:
1. **Reference** - Compare old vs new implementations
2. **Rollback** - Quick recovery if needed
3. **Learning** - Understand evolution of components

## Cleanup Policy

Files in this archive may be permanently deleted after:
- 3 months without issues in new implementation
- Confirmed no references in codebase
- Team approval

## Do Not Import

⚠️ **Warning:** These files should never be imported in the active codebase. They are for reference only.
